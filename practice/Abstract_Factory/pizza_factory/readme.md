## 例題
ピザを作ることを考えます。

- ピザは生地、ソース、トッピングからできます。
- 工場Aでは、生地に小麦、ソースにトマト、トッピングにコーンを使用します。
- 工場Bでは、生地に米粉、ソースにバジル、トッピングにチーズを使用します。
- すべての素材に確認のためのメソッド（check）をもたせます。

引用元に倣い、それぞれの工場のピザの内容についてコンソール出力する

（例）
工場A
Wheat(amount: 1.2)
Tomato(amount: 1.2)
Coan(amount: 1.2)

工場B
FlourDough(amount: 1.0)
Tomato(amount: 1.0)
Cheese(amount: 1.0)
